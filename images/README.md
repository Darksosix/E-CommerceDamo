## หมายเหตุ:
- หากไม่มีไฟล์รูปภาพ ระบบจะใช้รูปภาพจาก Unsplash อัตโนมัติ (fallback)

- รูปภาพควรมีพื้นหลังโปร่งใสหรือพื้นหลังสีขาวสำหรับผลลัพธ์ที่ดีที่สุด
